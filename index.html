1. 老闆薪資正式化

把利潤轉為合法薪資是節稅第一步

老闆不應只拿股利。正式領薪能將公司利潤轉化為「合法成本」。

職務配置：掛名總經理、技術長或專業顧問。
金流配置：固定月薪、年終、投保勞健保與提撥勞退。

2. SaaS 與雲端公司化

拒絕個人墊付，拿回 5% 營業稅

所有的軟體訂閱應全面由公司支付並打統編，不要讓老闆私人卡代刷。

OpenAI / AWS
Azure / GCP
GitHub / Figma
Slack / Notion
只要打上公司統編，就能扣抵 5% 營業稅並列為全額年度費用。

3. 設備汰換制度化

定期更新硬體，折舊可抵稅且戰力更強

定期更新開發戰力，同時產生合法折舊費用來抵銷盈餘。

  • MacBook / 高階工作站
  • NAS / 伺服器 / 網路設備
  • iPad / 測試用手機與平板
  • 建議兩年定期汰換，列報折舊

4. 差旅拜訪制度化

拜訪學校的每一塊錢都要變成本

賣校務系統跑學校是合理成本，應落實憑證紀錄，不可只剩模糊收據。

高鐵、Uber、停車費。
跨縣市住宿與客戶餐費。
⚠️ 必備三要素：拜訪紀錄、行程表、對應客戶名稱。

5. 研發費用最大化

軟體業特有的研發投抵，最高抵 15% 稅金

可列研發項目：

  • 工程師、UI/UX、PM 薪資
  • API 服務、AI 模型測試費
  • 外包開發、程式顧問費

關鍵：建立專案代碼

如 EDU-APP-2026。需對應 Git 紀錄、Jira 任務細節，未來申請抵減才容易通過。

6. 進階：公司分拆

透過分拆公司來隔離風險與優化毛利

業務規模化後，分拆公司能隔離標案風險並讓毛利更合理。

A 公司 (IP)

持有軟體著作權
收授權金

B 公司 (專案)

標案與客製化開發

C 公司 (維運)

資料庫與客服

7. 家人合法任職化

家人參與經營，合法分散所得稅負

配偶職責：

行政總務、人資或財務管理。需保留工作紀錄。

父母職責：

擔任技術顧問或行政支援。需定期支薪。

⚠️ 絕對重點：必須真實工作、定期薪轉與投保勞健保。禁止掛名。

8. 未分配盈餘規劃

不讓閒置資金被課 5% 盈餘稅

方案 A：發放股利

在個人所得與公司盈餘間找出最佳稅負點。

方案 B：再投資

開發 AI 新模組、購買伺服器，將利潤變資產。

9. 憑證與會計制度

找到懂「軟體與跨境」的會計師是關鍵

軟體業涉及 SaaS 授權與跨境費用，一般記帳士可能不熟悉。

雲端憑證：PDF發票、刷卡紀錄全面存檔。
分類明細：將雲端、差旅、廣告、研發正確分類。
